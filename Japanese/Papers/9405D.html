BWNo. : UGAI-94-05-D
題名 : FDTを利用したソフトウェア群の保守
著者 : 鵜飼孝典(UGAI, Takanori), 蓬莱尚幸(HORAI, Hisayuki)
出典 : 情報処理学会 第51回全国大会
連絡先 : ugai@fujitsu.com

概要 : 本稿では, ソフトウェア群をパッケージを単位として管理し, ユーザ側や開発 側からのアップデート要求に応じて必要に応じてパッケージを配布する枠組と システムを提案する. 本稿で提案する枠組は, ユーザ側と開発側の両側からの 要求に応じ, オンライン接続されない配布先もオンライン接続と同様に支援す る. さらに, ユーザ側の状況に応じた配布頻度などの設定が可能で, ソフトウ ェア群の仕様書からパッケージ間の依存関係などを生成し, 配布量の最適化な どを行なう. また本提案の枠組を形式仕様を用いて定義したことにより,オン ライン接続された配布先と, オンライン接続されない配布先の支援システムを 同じ定義からつくり出すことができ, システム自身の柔軟性, 拡張性を上げる ことができる.

概要 : In this paper, we propose an intelligent software package distributing system, which receives requirement from both developer and customer. The system handles cusomers which connects and does not connect with developer through network in a same framework. Farthermore the system controls software package distributing frequency in according to customer's requrement and optimize required software packages. We use the formal specification techniques on development the system, and the system has flexibility and extendability.
