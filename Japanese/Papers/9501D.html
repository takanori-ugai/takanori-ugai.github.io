BWNo. : UGAI-95-01-D
題名 : 形式仕様技術を利用したソフトウェア群の保守
著者 : 鵜飼孝典(UGAI, Takanori), 蓬莱尚幸(HORAI, Hisayuki), 中川徹(NAKAGAWA Toru), 高田裕志(TAKADA Yuji)
出典 : 情報処理学会 ソフトウェア工学研究会
連絡先 : ugai@fujitsu.com

概要 : 本稿では, ソフトウェア群をパッケージを単位として管理し, ユーザ側や開発側か らのアップデート要求に応じてパッケージを配布する枠組とシステムを提案す る. 本稿で提案する枠組は, ユーザ側と開発側の両側からの要求に応じ, オンライ ン接続されない配布先もオンライン接続された配布先と同様に支援する. さらに, ユーザ側の状 況に応じた配布頻度などの設定が可能で, ソフトウェア群の仕様書からパッケージ 間の依存関係などを生成し, 配布量の最適化などを行なう. オンライン接続された 配布先と, オンライン接続されない配布先の支援システムを同じ定義からつくり出 すことにより, システム自身の柔軟性, 拡張性を上げることができる.

概要 : In this paper, we propose an intelligent software package distributing system, which receives requirements from both developers and customers. The system provides services to customers whether they connect with developer through network or not. Furthermore the system controls software package distribution in according to customer's requirements and optimize the work for the distribution. The formal techniques allow us to developing the system with flexibility and extendibility.
