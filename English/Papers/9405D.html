Title : An Intelligent Software Package Distributing System with Formal Specification Techniques
Authors : UGAI Takanori, HORAI Hisayuki
Publisher: IPSJ Body
Language: Japanese

Abstract : In this paper, we propose an intelligent software package distributing system, which receives requirement from both developer and customer. The system handles cusomers which connects and does not connect with developer through network in a same framework. Farthermore the system controls software package distributing frequency in according to customer's requrement and optimize required software packages. We use the formal specification techniques on development the system, and the system has flexibility and extendability.
