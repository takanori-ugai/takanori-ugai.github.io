Title : An Intelligent Software Package Distributing System
Authors : Takanori Ugai, Hisayuki Horai, Toru Nakagawa, Yuji Takada
Published : IPSJ
Body
Language : Japanese Contact with : ugai@fujitsu.com

Abstract : In this paper, we propose an intelligent software package distributing system, which receives requirements from both developers and customers. The system provides services to customers whether they connect with developer through network or not. Furthermore the system controls software package distribution in according to customer's requirements and optimize the work for the distribution. The formal techniques allow us to developing the system with flexibility and extendibility.
