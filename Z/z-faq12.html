Z User Group ってなんですか?

Z User Group は 1992 年に Z に関する活動、とくに Z User Meeting を監督するため に作られました。comp.specification.z、 ZFORUM や郵便のメーリングリストへの参加 者としてあなた自身も Z User Group の一員であると考えることができます。現在のと ころ、メンバーには何の費用も与えられていませんが、必要とあらば見直すべき問題で あると考えています。詳しいことを知りたい人は <zforum-request@comlab.ox.ac.uk> と連絡を取って下さい。
